# 관련 연구 — 단일 RGB depth의 자세 적용 타당성

## 문서 요약

| 항목 | 내용 |
|---|---|
| 문서 유형 | 자세 적용 타당성 검토 |
| 적용 상태 | 근거 문서 |
| 다루는 범위 | depth 지표와 자세 판정 지표의 차이, 직접 근거의 범위 |
| 제품 내 역할 | 절대 측정 주장을 제한하고 자체 검증 필요성을 설명 |

## 1. “95% 정확도”는 depth 논문의 숫자가 아니다

단안 depth 논문이 보고하는 δ1은 예측 depth와 ground truth의 비율 오차가 1.25 이내인 픽셀 비율이다. 거북목 분류 accuracy, 머리 전방 거리의 ±5% 오차, 개인 baseline 대비 변화 탐지율과는 다른 지표다.

따라서 어떤 depth 벤치마크에서 δ1이 0.95 이상이어도 그 값을 제품 자세 정확도 95%의 근거로 사용하지 않는다.

## 2. 장면 평균 오차와 국소 차이 오차는 다르다

제품 신호는 같은 프레임의 머리 ROI와 몸통 ROI 사이 상대값이다. NYU·KITTI 같은 장면 벤치마크의 AbsRel은 전체 픽셀 평균 오차이므로 이 두 영역 차이의 오차를 직접 알려주지 않는다.

- 두 ROI의 오류가 같은 방향이면 차이에서 일부 상쇄될 수 있다.
- 머리카락·옷·배경 경계에서 오류가 다르면 차이가 더 흔들릴 수 있다.
- 카메라와 사람의 절대 거리에 AbsRel을 곱해 국소 차이 오차로 사용하는 계산은 성립하지 않는다.

올바른 결론은 “불가능”도 “충분히 정확”도 아니다. 공개 지표만으로 현재 자세 신호의 정확도를 결정할 수 없다는 것이다.

## 3. 인접 연구의 적용 범위

- PreventFHP는 공개 초록 기준으로 상용 depth 카메라를 사용한 하드웨어 depth 사례다. Kinect라는 기기명과 카메라-이마·몸통 거리 방식은 공개 초록이 아니라 논문 본문 또는 후속 문헌을 확인할 때만 별도로 인용한다.
- 단일 RGB에서 3D pose와 GCN을 사용한 FHP 연구는 약 78% 정확도를 보고했지만, 데이터·시점·모델이 현재 고정 정면 Mac 경로와 다르다.
- 측면 landmark 기반 연구는 82.4% 정확도를 보고했지만 표준화된 측면 촬영과 청소년 데이터에 기반한다.

이 연구들은 자세 자동화가 가능하다는 인접 근거일 뿐, DA-V2 relative depth 경로의 정확도 상한이나 하한이 아니다.

## 4. 현재 타당한 제품 범위

- 절대 cm 또는 임상 CVA를 출력하지 않는다.
- Vision 2D ROI와 DA-V2 relative depth로 개인 내 상대 feature를 만든다.
- 안내된 개인 baseline과 비교한다.
- 짧은 버스트와 품질 조건을 사용한다.
- 사전 정의한 자세 라벨로 accuracy·오경보·미탐·`noEval` 비율을 측정한다.

제품 데이터 검증 전에는 “95% 측정”을 주장하지 않는다.
