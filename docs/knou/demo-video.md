# 시연 영상 구성

제출용 시연 영상은 약 2분 30초로 구성한다. 과거 대회(2023·2024)의 1차 심사 결과 공지 기준 전시용 시연 영상 규격은 최대 3분이었다. 최종 작품 설명 동영상의 별도 규격이 안내되면 같은 원본을 이용해 길이와 해상도를 다시 맞춘다. 녹화, 편집, 내레이션과 내보내기의 도구별 절차는 [제작 실행 가이드](production-guide.md)를 따른다.

## 촬영 준비

- 메뉴 막대와 앱 화면이 읽히도록 화면을 확대한다.
- 알림 권한과 카메라 권한을 미리 확인한다.
- 바른 자세에서 기준 보정을 완료한다.
- 점검 주기를 15초로 설정한다.
- 알림 배너를 켠다.
- 개인 정보와 다른 앱의 알림이 보이지 않도록 정리한다.
- 카메라 촬영 당사자의 동의를 받고 불필요한 배경을 가린다.
- 같은 기능을 두 번 이상 촬영해 안정적인 장면을 선택한다.

## 장면 구성과 내레이션

| 시간 | 화면 | 내레이션 |
|---|---|---|
| 0:00~0:08 | 앱 아이콘과 작품명 | turtlemeck는 Mac 내장 카메라로 자세 습관을 점검하는 메뉴 막대 앱입니다. |
| 0:08~0:22 | 장시간 컴퓨터 작업 장면 | 화면에 집중하면 자세가 무너져도 바로 알아차리기 어렵고, 단순 타이머 알림은 실제 자세와 관계없이 사용자를 방해합니다. |
| 0:22~0:38 | 메뉴 막대 앱과 카메라 권한 | 앱은 별도 장비 없이 실행되며 최초 실행 시 카메라 권한만 요청합니다. |
| 0:38~0:55 | 기준 자세 보정 | 사용자가 바른 자세를 유지하면 개인 기준이 저장됩니다. 이후 모든 결과는 이 기준과 비교합니다. |
| 0:55~1:12 | 정상 상태와 다음 점검 시간 | 바른 자세에서는 정상 상태와 다음 점검 시간을 확인할 수 있습니다. |
| 1:12~1:40 | 앞으로 기운 자세, 연속 점검, 알림 | 머리가 앞으로 이동한 상태가 이어지면 앱은 일시적인 움직임인지 확인한 뒤 주의 상태와 알림을 표시합니다. |
| 1:40~1:57 | 자세 회복과 오늘 요약 | 자세를 바로잡으면 정상 상태로 돌아오며, 오늘의 자세 시간과 상태 변화가 로컬 통계에 반영됩니다. |
| 1:57~2:15 | 분석 흐름 그림 | 상체 위치와 상대 깊이를 함께 사용하고, 여러 장의 결과를 개인 기준과 비교합니다. 판단하기 어려운 입력은 나쁜 자세로 단정하지 않습니다. |
| 2:15~2:30 | 개인정보 안내와 마무리 | 분석은 기기 안에서 이루어집니다. 운영 모드에서는 이미지를 저장하거나 전송하지 않으며, 필요한 순간에만 자세를 떠올리게 합니다. |

## 필수 화면

- 앱 아이콘과 메뉴 막대 진입
- 기준 자세 보정
- 정상 상태
- 주의 상태와 알림
- 자세 회복
- 오늘 요약
- 점검 주기와 알림 설정
- 개인정보·비의료 안내

## 편집 기준

- 해상도는 1920×1080 이상을 권장한다.
- 메뉴 글자를 읽을 수 있도록 필요한 구간만 확대한다.
- 장면 전환 효과보다 실제 동작을 우선한다.
- 배경음악은 설명을 방해하지 않는 수준으로 사용하거나 생략한다.
- 실제 앱에서 확인하지 않은 성능 수치나 정확도는 넣지 않는다.
- 분석 결과가 잘 보이지 않는 장면은 자막으로 결과를 조작하지 말고 다시 촬영한다.
- 영상 마지막에 작품명, 참가자명과 연락처를 넣는다.

## 제출 전 확인

- 음성과 자막의 설명이 실제 동작과 일치함
- 의료 진단이나 정확한 거리 측정으로 오해할 표현이 없음
- 촬영 이미지가 외부 전송되지 않는다는 설명이 운영 모드 기준임
- 제3자 이미지, 음원과 글꼴을 사용했다면 사용 조건을 확인함
- 영상 파일이 다른 Mac에서도 재생됨
- 참가자명과 연락처가 정확함
