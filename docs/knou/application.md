## 작품 개요

다음 문안을 작품 개요 칸에 사용한다.

turtlemeck는 장시간 컴퓨터를 사용하는 사람의 자세 습관 형성을 돕는 macOS 메뉴 막대 앱이다. 내장 카메라로 짧은 이미지 묶음을 촬영하고, 상체 위치와 머리·몸통의 상대적인 앞뒤 관계를 기기 안에서 분석한다. 최초 실행 시 사용자의 바른 자세를 기준으로 저장하고 이후 결과를 개인 기준과 비교한다. 일시적인 움직임이나 분석하기 어려운 화면은 나쁜 자세로 단정하지 않으며, 좋지 않은 상태가 계속될 때만 알림을 제공한다. 운영 모드에서는 촬영 이미지를 저장하거나 외부로 전송하지 않는다. 자세 상태, 오늘의 바른 자세·주의 자세 시간과 알림 횟수를 메뉴 막대에서 간단히 확인할 수 있다.

## 교과목 응용 내용

실제로 수강하고 작품에 적용한 교과목만 작성한다. 다음 형식으로 한두 문단을 구성한다.

`[교과목명]`에서 학습한 `[개념 또는 기법]`을 `[작품의 기능]`에 적용했다. 구체적으로 `[입력 처리, 알고리즘, 상태 관리, 사용자 인터페이스 등 실제 적용 내용]`을 설계하고 구현하는 데 활용했다. 이를 통해 `[사용자에게 제공되는 결과 또는 개선점]`을 구현했다.

작성할 때 단순히 기술 이름을 나열하지 않는다. 수업에서 배운 내용과 실제 기능의 연결 관계를 설명한다.

## 오픈소스와 라이선스

다음 문안을 출처 칸에 사용한다. 칸이 부족하면 핵심 내용만 적고 `ThirdPartyNotices.md`를 별도 첨부한다.

- Apple PoseNet MobileNet 0.75 모델, Apache-2.0, https://developer.apple.com/documentation/coreml/detecting-human-body-poses-in-an-image
- Apple PoseNet 샘플 코드 사용 부분, MIT
