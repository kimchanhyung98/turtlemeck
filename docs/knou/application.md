## 작품 개요

다음 문안을 작품 개요 칸에 사용한다.

turtlemeck는 장시간 컴퓨터를 사용하는 사람의 자세 습관 형성을 돕는 macOS 메뉴 막대 앱이다. 내장 카메라로 짧은 이미지 묶음을 촬영하고, 상체 위치와 머리·몸통의 상대적인 앞뒤 관계를 기기 안에서 분석한다. 최초 실행 시 사용자의 바른 자세를 기준으로 저장하고 이후 결과를 개인 기준과 비교한다. 일시적인 움직임이나 분석하기 어려운 화면은 나쁜 자세로 단정하지 않으며, 좋지 않은 상태가 계속될 때만 알림을 제공한다. 운영 모드에서는 촬영 이미지를 저장하거나 외부로 전송하지 않는다. 자세 상태, 오늘의 바른 자세·주의 자세 시간과 알림 횟수를 메뉴 막대에서 간단히 확인할 수 있다.

## 개발 배경

다음 문안을 작품 개요의 개발 배경 설명에 사용한다.

노트북 화면만 들여다보는 생활로 팀원 스스로 목과 어깨 건강이 나빠지는 것을 체감했고, 최근 가족(삼촌)이 목디스크로 수술을 받으면서 바른 자세 습관을 도와주는 도구의 필요성을 절감했다. 가족과 주변 사람들이 모두 MacBook을 사용하는 환경을 고려해, 모든 노트북에 기본으로 있는 웹캠 하나만으로 동작하는 macOS 전용 앱으로 개발했다.

## 교과목 응용 내용

다음 문안을 수업 내용 응용 설명에 사용한다. 팀원 모두 2026학년도 1학기에 인공지능 과목을 수강했고, 김찬형은 2학기에 머신러닝·딥러닝 과목을 수강 신청했다.

팀원 모두 2026학년도 1학기에 수강한 인공지능 과목에서 학습한 기계학습·신경망의 기본 개념과 모델 추론·평가 관점을, 2D 자세 추정(PoseNet)과 단안 상대 깊이 추정(Depth Anything V2 Small) 모델의 선정·결합과 Core ML 온디바이스 실행에 적용했다. 모델 출력을 그대로 신뢰하지 않고 정렬 기반 순위 통계(중앙값·백분위수·사분위범위)와 상태 기계 설계로 여러 프레임의 대표값 집계, 불안정한 입력 제외와 정상·주의·판단 보류 상태 전이를 구성했다. 개발 과정에서 인공지능 분야에 관심이 깊어져 팀장이 2학기 머신러닝·딥러닝 과목을 수강 신청했다.
